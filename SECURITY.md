# Security Notes

## Sensitive data

Do not commit:

- `.env` files
- database files
- private keys or certificates
- runtime exports or backups from `var/`

## Local development

Use the example env files in the current `115 Weightlifting` app directory:

- `115-weightlifting/src/backend/.env.example`
- `115-weightlifting/src/frontend/.env.example`

## Reporting

For this private repository, security issues should be handled directly and not discussed in public issue trackers if the repo is ever made public later.
