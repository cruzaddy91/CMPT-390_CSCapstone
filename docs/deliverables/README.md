# Deliverables

Turn-in materials for the capstone. One folder per deliverable type.

## Structure

| Folder | Contents |
|--------|----------|
| **proposals/** | Full proposal (Markdown + Word). Single source for what you submit. |
| **proof_of_concept/** | Proof of concept summary (1 to 2 page): .md, .txt, .docx. |
| **reports/** | LaTeX report source (Weightlifting_Platform_Report.tex, template Report.tex) and compile instructions. |
| **transcripts/** | AI session transcripts (Markdown, PDF exports, summary, paraphrase). |
| **other/** | Other submitted PDFs (i.e., About me, Post Graduation Plans, Brainstorming Notes). |

## Quick reference

- **Proposal (submit):** `proposals/project_proposal.md`, `proposals/Full_Proposal_Final.docx`
- **Proof of concept:** `proof_of_concept/proof_of_concept.md`, `proof_of_concept/proof_of_concept.txt`, `proof_of_concept/proof_of_concept.docx`
- **Report source:** `reports/Weightlifting_Platform_Report.tex`
- **Transcripts:** `transcripts/Transcript_1_Architecture_Planning.*`, `Transcript_2_Data_Modeling_Implementation.*`
