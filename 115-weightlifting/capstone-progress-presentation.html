What we’ve completed

  • Programs API: GET /api/programs/ — returns programs for the logged-in user (coach or athlete)
  • Database models and migrations for User (coach/athlete), TrainingProgram, and related apps
  • CORS configured so the React app can call the API
Frontend (React)
  • Login page: username/password, calls API, stores token
  • Register: create account with role (coach or athlete)
  • Protected routes: Coach and Athlete dashboards require login; otherwise redirect to login
  • Coach Dashboard calls GET /api/programs/ with the token and shows “Programs from API: N”
  • Log out in the nav when logged in
